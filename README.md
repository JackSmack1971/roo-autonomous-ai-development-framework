- **MCP Services** configured (Exa, Perplexity, Context7 recommended)
- **Auto-approval** settings enabled in Roo Code UI

### 30-Second Setup

```bash

git clone <repository-url>
cd roo-autonomous-ai-development-framework
code .

# 2. Activate enhanced modes

# 3. Configure auto-approval in Roo Code UI:
#    ✅ Always approve mode switching

#    ✅ Always approve quality interventions

# 4. Start autonomous development
```
